## 3. 戦略設計

### 3.1 設計原理

特性分析の発見を統合した**適応型多因子戦略**:

1. **VR<1の平均回帰性** → Z-score逆張り（30日窓、±1.5エントリー）
2. **H>0.55のトレンド性** → ドンチャンブレイクアウト（20日高値/安値）
3. **レジーム効果** → JPYクロスにリスクオン/オフフィルター適用
4. **相関制約** → 同時ポジション間の相関0.6未満を要求

### 3.2 ウォークフォワード手法

訓練期間（5年）でペアごとに最適な戦略・パラメータをSharpe比で選定し、テスト期間（1年）でOOS検証。22期間のローリングテスト。

### 3.3 ポジション管理

| 制約 | 値 |
|------|-----|
| 最大同時ポジション | 8 |
| 最大通貨エクスポージャー | 同一通貨3ポジション |
| 相関制限 | 既存ポジションとの60日相関 < 0.6 |
| リスクサイズ | 1トレード3%（2%vol基準でスケーリング） |
| JPYレジームフィルター | リスクオフ時はJPYロング禁止、リスクオン時はJPYショート禁止 |

## 5. 結論

### 5.1 主要発見

1. **全通貨ペアに日次トレンド性（H>0.55）と中期平均回帰性（VR<1）の二重構造がある**
   - これがZ-score逆張り戦略の根本的な優位性の源泉
   - 30日窓のZ-scoreは中期平均回帰性をうまく捕捉

2. **EUR/GBPが最も安定したペア**（勝率77.1%、PF 1.86）
   - ユーロとポンドの高い経済的連動性が安定した平均回帰を保証

3. **レジームフィルターでJPYクロスのリスクを低減**
   - リスクオフ時のJPYロング回避で大幅DDを軽減

4. **相関フィルターが過度なリスク集中を防止**
   - JPYクロス間は相関0.72~0.89 → 同時保有制限が効果的

### 5.2 リスクと限界

- **最大DD 48.32%**は大きい。実運用ではレバレッジを抑制すべき
- 日次終値データのみ使用。日中の変動（ストップロス狩り）は反映されていない
- ECBデータのため**スプレッド・スリッページ未考慮**
- ZAR/JPYはデータなしで分析除外
- 2008年（リーマン）、2016年（Brexit）に大きなDD。テールリスクイベントへの耐性は限定的

### 5.3 実運用への提言

1. **上位ペアに集中**: EUR/GBP, USD/CAD, AUD/CAD, EUR/CAD（Z-score、PF>1.3）
2. **レバレッジ制限**: MaxDD 48%を考慮し、レバ5倍以下を推奨
3. **USD/JPYドンチャンは除外検討**: PF 0.74で明確に不利
4. **月次リバランス**: 訓練期間のパラメータを四半期ごとに更新
5. **リスクオフ時はポジション縮小**: JPYインデックスベースのリスク管理
