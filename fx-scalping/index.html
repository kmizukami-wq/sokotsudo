退出ルール（ユーザー仕様）
peak = max(過去のpip損益) とする。
例1: +2 → +1 → 利確（ピーク2から下落、利益確保）
例2: +2 → +4 → +3 → 利確（ピーク4から下落、伸ばした分を確保）
例3: +2 → -1 → 即損切（マイナス転落）
ロジック: peak > 0 かつ 現pip < peak → 利確 ／ 現pip < 0 → 即損切 ／ 60秒タイムアウト
